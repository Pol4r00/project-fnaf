=INSTRUCTIONS=

1- Make sure you have python installed
2- Open powershell or any other terminal emulator
3- Run "pip install pygame"
4- Still in the terminal, type in the path where the game files are located, and then run python3 main.py

=Controls=

I - Show animatronic locations
U - Show hitboxes
Slide mouse from left to right to move the camera
Slide the mouse down to open the cameras
